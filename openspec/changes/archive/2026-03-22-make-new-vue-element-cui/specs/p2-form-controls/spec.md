## 新增需求

### 需求：CuiSelect 实现

CuiSelect 应当提供一个增强的选择器组件，封装 Element Plus 的 ElSelect。

#### 场景：选项渲染

- **当** CuiSelect 接收到选项数组时
- **则** 应当在下拉菜单中渲染所有选项

#### 场景：值绑定

- **当** modelValue 发生变化时
- **则** 应当更新选中的值并触发 update:modelValue 事件

#### 场景：远程搜索

- **当** remote 属性为 true 时
- **则** 应当支持通过 remote-method 进行远程数据获取

### 需求：CuiSelectEnum 实现

CuiSelectEnum 应当提供一个专门用于枚举值选择的选择器组件。

#### 场景：枚举选项

- **当** CuiSelectEnum 接收到枚举对象时
- **则** 应当从枚举的键和值生成选项

#### 场景：枚举值绑定

- **当** 用户选择枚举选项时
- **则** 应当触发枚举值（而非标签）

### 需求：CuiSelectDict 实现

CuiSelectDict 应当提供一个基于字典的选项选择器组件。

#### 场景：字典加载

- **当** 提供 dictCode 属性时
- **则** 应当从 API 获取字典选项

#### 场景：字典缓存

- **当** 多次使用相同的 dictCode 时
- **则** 应当缓存字典数据以避免冗余请求

### 需求：CuiRadioEnum 实现

CuiRadioEnum 应当提供一个用于枚举值选择的单选按钮组组件。

#### 场景：枚举单选按钮渲染

- **当** CuiRadioEnum 接收到枚举对象时
- **则** 应当为每个枚举值渲染单选按钮

#### 场景：单选按钮选择

- **当** 用户选择单选按钮选项时
- **则** 应当触发枚举值

### 需求：CuiRadioDict 实现

CuiRadioDict 应当提供一个基于字典的单选按钮组组件。

#### 场景：字典单选按钮渲染

- **当** CuiRadioDict 接收到 dictCode 时
- **则** 应当获取并渲染字典选项的单选按钮

### 需求：CuiCheckboxEnum 实现

CuiCheckboxEnum 应当提供一个用于多选枚举值的复选框组组件。

#### 场景：枚举复选框渲染

- **当** CuiCheckboxEnum 接收到枚举对象时
- **则** 应当为每个枚举值渲染复选框

#### 场景：多选

- **当** 用户选择多个复选框时
- **则** 应当触发选中的枚举值数组

### 需求：CuiAutocomplete 实现

CuiAutocomplete 应当提供一个支持建议的自动完成输入组件。

#### 场景：建议显示

- **当** 用户在输入框中输入时
- **则** 应当显示包含匹配选项的建议下拉菜单

#### 场景：远程建议

- **当** 提供 fetch-suggestions 属性时
- **则** 应当调用函数异步获取建议

#### 场景：建议选择

- **当** 用户选择建议时
- **则** 应当更新输入值并触发 select 事件
