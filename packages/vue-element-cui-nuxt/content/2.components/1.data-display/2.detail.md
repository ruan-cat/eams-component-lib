# CuiDetail

`CuiDetail` 用于展示结构化详情信息，适合用户信息、工单信息和审批记录等场景。

#code

```vue
<script setup lang="ts">
import { CuiDetail } from "@eams-monorepo/vue-element-cui";
import type { CuiDetailField } from "@eams-monorepo/vue-element-cui";

const fields: CuiDetailField[] = [
  { label: "资产名称", prop: "name" },
  { label: "归属部门", prop: "dept" },
  { label: "资产状态", prop: "status", render: (row) => row.status === 1 ? "在用" : "闲置" },
]
const detail = { name: "固定资产台账", dept: "财务共享中心", status: 1 }
</script>

<template>
  <CuiDetail :fields="fields" :data="detail" :columns="2" />
</template>
```

| `labelWidth` | `string` | `'140px'` | 标签宽度 |

:::api-table{title="CuiDetailField 字段配置"}
| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `label` | `string` | 标签文字（必填） |
| `prop` | `string` | 数据字段名（必填） |
| `width` | `string` | 字段宽度（可选，用于跨列展示） |
| `render` | `(row: any) => any` | 自定义渲染函数（取代插槽） |
:::

:::doc-note{title="使用 render 函数自定义渲染" tone="info"}
CuiDetail 不提供 Slots，复杂的单元格渲染通过 `render` 函数实现：

\`\`\`vue
const fields = [
  { label: '资产状态', prop: 'status', render: (row) => row.status === 1 ? '在用' : '闲置' }
]
\`\`\`
:::

## 推荐写法

```vue
<script setup lang="ts">
import { CuiDetail } from "@eams-monorepo/vue-element-cui";
import type { CuiDetailField } from "@eams-monorepo/vue-element-cui";

const fields: CuiDetailField[] = [
	{ label: "资产名称", prop: "name" },
	{ label: "负责人", prop: "owner" },
	{ label: "状态", prop: "status", render: (row) => (row.status === 1 ? "启用" : "停用") },
];

const detail = {
	name: "固定资产台账",
	owner: "财务共享中心",
	status: 1,
};
</script>

<template>
	<CuiDetail :fields="fields" :data="detail" />
</template>
```

:::doc-note{title="CuiDetail 不发出事件" tone="info"}
此组件没有任何 Emits。所有定制展示逻辑通过 `CuiDetailField.render` 函数实现，无需监听事件。
:::
