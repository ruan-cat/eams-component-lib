# CuiExcel

`CuiExcel` 负责把上传、模板下载和结果反馈整合到统一流程里，适合数据导入场景。

#code

```vue
<script setup lang="ts">
import { ref } from "vue";
import { CuiExcel } from "@eams-monorepo/vue-element-cui";

const showDialog = ref(false);

const handleSuccess = (response: any, file: any) => {
  console.log("上传成功", file.name);
  showDialog.value = false;
};
</script>

<template>
  <el-button type="primary" @click="showDialog = true">打开导入</el-button>
  <CuiExcel
    v-model="showDialog"
    upload-url="/api/assets/import"
    template-url="/api/assets/template"
    title="导入资产台账"
    @success="handleSuccess"
  />
</template>
```

::api-table{title="核心 Props" description="导入流程最常用配置"}
| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `modelValue` | `boolean` | 弹层是否打开 |
| `uploadUrl` | `string` | 上传地址 |
| `templateUrl` | `string` | 模板下载地址 |
| `headers` | `Record<string, string>` | 请求头 |
| `params` | `Record<string, any>` | 额外参数 |
::

::api-table{title="Props"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `boolean` | — | 是否显示，支持 v-model（必填） |
| `uploadUrl` | `string` | — | 文件上传接口 URL（必填） |
| `title` | `string` | `'导入 Excel'` | 对话框标题 |
| `templateUrl` | `string` | — | 模板文件下载 URL |
| `headers` | `Record<string, string>` | `{}` | 上传请求头（如 Authorization） |
| `params` | `Record<string, any>` | `{}` | 上传时附带的额外参数 |
| `accept` | `string` | `'.xls,.xlsx'` | 接受的文件格式 |
::

::api-table{title="Emits"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `(value: boolean)` | 显示状态更新 |
| `success` | `(response, file, fileList)` | 文件上传成功 |
| `error` | `(error: any)` | 文件上传失败 |
::
