::api-table{title="核心 Props" description="最常用的表格配置入口"}
| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `data` | `T[]` | 表格数据数组 |
| `columns` | `CuiTableColumn<T>[]` | 列配置数组 |
| `loading` | `boolean` | 是否显示加载状态 |
| `showSelection` | `boolean` | 是否开启多选列 |
| `showIndex` | `boolean` | 是否开启序号列 |
::

::doc-note{title="使用建议" tone="info"}
列表页优先从 `CuiTable` 开始。复杂展示需求尽量收敛在列配置与插槽，不要在页面层重复搭表格骨架。
::
