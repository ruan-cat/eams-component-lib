## 阅读顺序

1. 先看 [表格](/components/data-display/table)，建立列表页基础模型。
2. 再看 [详情](/components/data-display/detail)，处理字段与展示布局。
3. 最后看 [Excel 导入](/components/data-display/excel)，补齐导入流程说明。
