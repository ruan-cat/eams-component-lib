# CuiSelectDict

`CuiSelectDict` 是面向字典数据的选择框，通过 `dictCode` 从字典服务自动加载候选项，并在内部维护字典缓存，避免重复请求。

#code

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { CuiSelectDict } from '@eams-monorepo/vue-element-cui'

const selected = ref<string | null>(null)
</script>

<template>
  <CuiSelectDict v-model="selected" dict-code="DEPT_TYPE" placeholder="请选择部门类型" />
</template>
```

::

::api-table{title="Props" description="CuiSelectDict 配置项"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `any` | — | 当前选中值，支持 v-model（必填） |
| `dictCode` | `string` | — | 字典编码，用于从字典服务加载选项（必填） |
| `placeholder` | `string` | `'请选择'` | 占位文本 |
| `disabled` | `boolean` | `false` | 是否禁用 |
| `clearable` | `boolean` | `true` | 是否可清空 |
| `filterable` | `boolean` | `true` | 是否可过滤搜索 |
::

::api-table{title="Emits" description="CuiSelectDict 事件"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `value: any` | 选中值变化时触发，用于 v-model 绑定 |
| `change` | `value: any` | 选中值变化时触发 |
::

::doc-note{tone="warning" title="字典缓存"}
组件内部维护一个字典数据缓存 Map，相同 `dictCode` 的数据只请求一次。实际使用时需要对接字典服务，确保字典服务已正确配置，`dictCode` 对应的字典数据已录入。缓存仅在组件生命周期内有效，页面刷新后重新拉取。
::
