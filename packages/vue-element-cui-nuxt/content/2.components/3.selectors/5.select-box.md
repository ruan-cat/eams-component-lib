# CuiSelectBox

`CuiSelectBox` 是弹层式选择框，通过 `selector` 插槽挂载自定义选择内容（如人员列表、树形选择器等），适用于候选项复杂、需要确认操作的场景。

#code

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { CuiSelectBox } from '@eams-monorepo/vue-element-cui'

const selected = ref<string | null>(null)
</script>

<template>
  <CuiSelectBox v-model="selected" type="user" placeholder="请选择负责人">
    <template #selector="{ condition, onSubmit }">
      <!-- 此处放置自定义选择器内容 -->
      <el-button type="primary" @click="onSubmit('user-001')">选择张三</el-button>
    </template>
  </CuiSelectBox>
</template>
```

::api-table{title="Props" description="CuiSelectBox 配置项"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `string \| string[]` | — | 当前选中值，支持 v-model（必填） |
| `type` | `string` | — | 选择器类型标识，用于区分不同业务场景（必填） |
| `limit` | `number` | `0` | 最大可选数量，0 表示不限制 |
| `disabled` | `boolean` | `false` | 是否禁用 |
| `condition` | `Record<string, any>` | `{}` | 传入弹层的筛选条件，会同步到 selector 插槽的 condition 参数 |
| `placeholder` | `string` | `'请选择'` | 占位文本 |
::

::api-table{title="Emits" description="CuiSelectBox 事件"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `value: string \| string[]` | 选中值变化时触发，用于 v-model 绑定 |
| `submit` | `value: string \| string[]` | 弹层确认提交时触发 |
::

::api-table{title="Slots" description="CuiSelectBox 插槽"}
| 插槽名 | 参数 | 说明 |
| --- | --- | --- |
| `selector` | `{ condition, onSubmit }` | 弹层内的选择器内容，需自行实现。`condition` 为当前筛选条件，`onSubmit(value)` 用于提交选中结果并关闭弹层 |
::
