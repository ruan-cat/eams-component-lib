# CuiRadioDict

`CuiRadioDict` 用于从字典服务加载选项并渲染单选按钮组。

::demo-playground
---
title: 基础用法
description: 演示 CuiRadioDict 的基础用法
---

#preview
<RadioDictBasicDemo />

#code

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { CuiRadioDict } from '@eams-monorepo/vue-element-cui'

const selected = ref<string | null>(null)
</script>

<template>
  <CuiRadioDict v-model="selected" dict-code="DEPT_TYPE" />
</template>
```

::

::api-table{title="Props"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `any` | — | 当前选中值，支持 v-model，必填 |
| `dictCode` | `string` | — | 字典编码，用于从字典服务获取选项，必填 |
| `disabled` | `boolean` | `false` | 是否禁用整组 |
::

::api-table{title="Emits"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `value: any` | 选中值变化时触发，配合 v-model 使用 |
| `change` | `value: any` | 用户手动切换选中项时触发 |
::

::doc-note{tone="warning" title="字典缓存说明"}
组件内部维护字典缓存 Map，相同 `dictCode` 在同一页面生命周期内只请求一次。实际使用时需要接入真实的字典服务，确保字典编码与后端保持一致。
::
