# CuiForm

`CuiForm` 负责录入与编辑场景的主表单能力，重点在字段配置驱动、统一验证和可复用布局。

::api-table{title="核心 Props" description="表单最常用配置"}
| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `fields` | `CuiFormField[]` | 字段配置 |
| `modelValue` | `Record<string, any>` | 表单数据 |
| `labelWidth` | `string | number` | 标签宽度 |
| `inline` | `boolean` | 是否行内布局 |
| `disabled` | `boolean` | 是否禁用整个表单 |
::

::api-table{title="完整 Props" description="所有可配置属性"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `fields` | `CuiFormField[]` | — | 字段配置数组（必填） |
| `modelValue` | `T` | — | 表单数据，支持 v-model（必填） |
| `labelWidth` | `string \| number` | `'100px'` | 标签宽度 |
| `inline` | `boolean` | `false` | 是否行内布局 |
| `disabled` | `boolean` | `false` | 是否禁用整个表单 |
| `labelPosition` | `'left' \| 'right' \| 'top'` | `'right'` | 标签对齐方式 |
::

::api-table{title="CuiFormField 字段配置"}
| 属性 | 类型 | 说明 |
| --- | --- | --- |
| `prop` | `string` | 字段名（必填） |
| `label` | `string` | 标签文字（必填） |
| `type` | `'input' \| 'select' \| 'date' \| 'textarea' \| 'number'` | 字段类型 |
| `placeholder` | `string` | 占位符文字 |
| `required` | `boolean` | 是否必填 |
| `rules` | `FormItemRule[]` | 自定义校验规则 |
| `options` | `{ label: string; value: any }[]` | select 类型的选项列表 |
| `disabled` | `boolean` | 是否禁用该字段 |
| `defaultValue` | `any` | 字段默认值 |
| `slot` | `string` | 自定义插槽名（配合插槽使用） |
::

::api-table{title="Emits"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `(value: T)` | 表单数据更新 |
| `submit` | `(value: T)` | 表单验证通过后提交 |
| `reset` | `()` | 重置表单 |
::

::api-table{title="Slots"}
| 插槽名 | Scope | 说明 |
| --- | --- | --- |
| `[field.slot]`（动态名） | `{ field: CuiFormField, value: any, updateValue: (v: any) => void }` | 当字段配置了 slot 属性时，通过该插槽自定义渲染 |
::

::api-table{title="Expose（ref 方法）"}
| 方法 | 类型 | 说明 |
| --- | --- | --- |
| `validate` | `() => Promise<CuiFormValidateResult>` | 手动触发表单验证 |
| `resetFields` | `() => void` | 重置所有字段 |
::
