# CuiAutocomplete

`CuiAutocomplete` 用于联想输入场景，支持静态建议列表和远程建议加载。

<FeatureList
	:items='[
		{ "title": "有限候选项优先", "description": "适合资产名称、编码、部门等有限候选项的联想输入。" },
		{ "title": "支持异步建议", "description": "复杂搜索场景可通过 fetchSuggestions 透出异步建议能力。" },
		{ "title": "不替代复杂选择", "description": "联想只负责缩短输入路径，不应替代明确筛选或弹层选择。" }
	]'
/>

::api-table{title="Props" description="自动完成完整 Props 配置"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `string` | — | 当前输入值，支持 v-model，必填 |
| `suggestions` | `CuiAutocompleteSuggestion[]` | `[]` | 静态建议列表 |
| `fetchSuggestions` | `(queryString: string, cb: Function) => void` | — | 异步获取建议的方法，与 suggestions 二选一 |
| `placeholder` | `string` | `'请输入'` | 输入框占位文字 |
| `disabled` | `boolean` | `false` | 是否禁用 |
| `clearable` | `boolean` | `true` | 是否可清空 |
| `debounce` | `number` | `300` | 输入防抖延迟（毫秒） |
| `triggerOnFocus` | `boolean` | `true` | 聚焦时是否立即展示建议 |
::

::api-table{title="Emits"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `value: string` | 输入值变化时触发，配合 v-model 使用 |
| `select` | `item: CuiAutocompleteSuggestion` | 用户从候选项中选中某一项时触发 |
::

::doc-note{tone="info" title="CuiAutocompleteSuggestion 类型"}
建议项类型定义为 `{ value: string; label?: string; [key: string]: any }`，其中 `value` 为必填项，`label` 可选用于显示不同于 value 的文本。
::

## 示例

```vue
<script setup lang="ts">
import { ref } from "vue";
import { CuiAutocomplete } from "@eams-monorepo/vue-element-cui";

const keyword = ref("");
const suggestions = [{ value: "固定资产台账" }, { value: "库存物料盘点" }];
</script>

<template>
	<CuiAutocomplete v-model="keyword" :suggestions="suggestions" placeholder="请输入关键词" />
</template>
```
