# CuiRadioEnum

`CuiRadioEnum` 用于从枚举服务加载选项并渲染单选按钮组。

::demo-playground
---
title: 基础用法
description: 演示 CuiRadioEnum 的基础用法
---

#preview
<RadioEnumBasicDemo />

#code

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { CuiRadioEnum } from '@eams-monorepo/vue-element-cui'

const selected = ref<string | null>(null)
</script>

<template>
  <CuiRadioEnum v-model="selected" enum-code="ASSET_STATUS" />
</template>
```

::

::api-table{title="Props"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `any` | — | 当前选中值，支持 v-model，必填 |
| `enumCode` | `string` | — | 枚举编码，用于从枚举服务获取选项，必填 |
| `disabled` | `boolean` | `false` | 是否禁用整组 |
::

::api-table{title="Emits"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `value: any` | 选中值变化时触发，配合 v-model 使用 |
| `change` | `value: any` | 用户手动切换选中项时触发 |
::

::doc-note{tone="warning" title="选项加载说明"}
Demo 中使用的枚举编码（如 `ASSET_STATUS`）为 mock 数据。实际使用时需要接入真实的枚举服务，确保枚举编码与后端保持一致。
::
