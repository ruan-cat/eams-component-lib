# CuiCheckboxEnum

`CuiCheckboxEnum` 用于从枚举服务加载选项并渲染多选按钮组。

::demo-playground
---
title: 基础用法
description: 演示 CuiCheckboxEnum 的基础用法
---

<CheckboxEnumBasicDemo />

#code

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { CuiCheckboxEnum } from '@eams-monorepo/vue-element-cui'

const selected = ref<string[]>([])
</script>

<template>
  <CuiCheckboxEnum v-model="selected" enum-code="ASSET_TYPE" />
</template>
```

::

::api-table{title="Props"}
| 属性 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `modelValue` | `any[]` | — | 当前选中值数组，支持 v-model，必填 |
| `enumCode` | `string` | — | 枚举编码，用于从枚举服务获取选项，必填 |
| `disabled` | `boolean` | `false` | 是否禁用整组 |
::

::api-table{title="Emits"}
| 事件名 | 参数 | 说明 |
| --- | --- | --- |
| `update:modelValue` | `value: any[]` | 选中值数组变化时触发，配合 v-model 使用 |
| `change` | `value: any[]` | 用户手动切换选中项时触发 |
::

::doc-note{tone="info" title="多选与 v-model"}
`modelValue` 为数组类型，请确保绑定值初始化为数组（如 `ref([])`）。v-model 双向绑定时，每次选中变化会返回完整的新数组，而非差量更新。
::
